# Portfolio Pribadi - Raden Manis

Website portfolio pribadi dengan desain modern dan responsif yang menampilkan CV, pengalaman kerja, keahlian, dan portfolio proyek.

## Fitur

- Desain responsif dan modern
- Navigasi smooth scroll
- Animasi skill bars
- Filter portfolio
- Form kontak dengan integrasi FormSubmit
- Optimasi untuk semua perangkat

## Penggunaan

1. Ganti informasi pribadi di file `index.html`:
   - Nama, judul, deskripsi
   - Pengalaman kerja
   - Keahlian
   - Portfolio proyek
   - Informasi kontak

2. Ganti gambar:
   - `assets/images/profile.jpg` dengan foto profil Anda
   - `assets/images/project1.jpg`, `project2.jpg`, dll dengan gambar proyek Anda

3. Untuk form kontak, sudah dikonfigurasi untuk mengirim ke email `radenmanis123@gmail.com` melalui FormSubmit.

## Deploy ke Vercel

1. Buat repository GitHub dan push semua file
2. Login ke [Vercel](https://vercel.com)
3. Import project dari GitHub
4. Konfigurasi deployment (Vercel akan otomatis mendeteksi sebagai project static)
5. Klik Deploy

## Customisasi

- Warna: Ubah variabel CSS di `:root` di file `css/style.css`
- Konten: Edit langsung di `index.html`
- Styling: Sesuaikan di `css/style.css`
- Interaktivitas: Edit di `js/script.js`

## Teknologi

- HTML5
- CSS3 (dengan variabel CSS dan Flexbox/Grid)
- JavaScript (ES6+)
- Font Awesome untuk ikon
- Google Fonts (Poppins)
